ML Internship Tasks

Overview
This repository contains the completed tasks assigned as part of an ML Internship selection process.
Each task is implemented as an independent project with proper structure, documentation, and error handling.

Repository Structure
- testtospeech
  Text-to-Speech web application built using Python and Streamlit.
  Demonstrates UI design, input validation, audio generation, and testing.

- deepseekapiintegration
  AI-powered chat assistant built using Python and Streamlit.
  Demonstrates API integration, session-based conversation handling, logging, and graceful error handling.

Key Highlights
- Clean and modular project structure
- Proper use of virtual environments (excluded from version control)
- Robust error handling and logging
- Professional UI using Streamlit with custom styling
- Clear documentation for each task

Technologies Used
- Python
- Streamlit
- REST APIs
- Git and GitHub

How to Run
Each project folder contains its own README with step-by-step instructions to set up and run the application locally.
